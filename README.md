Calculated customer retention rates by:
1. Grouping customers by acquisition cohort and month index.
2. Creating a cohort matrix using a pivot table.
3. Dividing each cohort's customer counts by its initial size to obtain retention rates.

```python
# Group by Cohort Month and Cohort Index
cohort_data = (
    df.groupby(['CohortMonth', 'CohortIndex'])['Customer ID']
      .nunique()
      .reset_index()
)

# Pivot the Data to a Table
cohort_counts = cohort_data.pivot(
    index='CohortMonth',
    columns='CohortIndex',
    values='Customer ID'
)

# Calculate Retention Rates
cohort_size = cohort_counts.iloc[:, 0]

retention = cohort_counts.divide(
    cohort_size,
    axis=0
)
```

### Customer Segmentation (RFM)

* Calculated Recency, Frequency, and Monetary metrics for each customer.
* Applied quintile-based scoring to create RFM scores.
* Segmented customers into:
  * Champions
  * Loyal Customers
  * Potential Loyalists
  * At Risk
  * Lost Customers
* Used segmentation to support retention and marketing strategies.

<details>
<summary><b>Click to View Python: Customer Segmenting</b></summary>

```python
# Calculate snapshot date
snapshot_date = df["InvoiceDate"].max() + pd.Timedelta(days=1)

# Calculate RFM metrics
rfm = (
    df.groupby("Customer ID")
      .agg({
          "InvoiceDate": lambda x: (snapshot_date - x.max()).days,
          "Invoice": "nunique",
          "Revenue": "sum"
      })
      .reset_index()
)

rfm.columns = [
    "CustomerID",
    "Recency",
    "Frequency",
    "Monetary"
]

# Assign RFM scores
rfm["R_Score"] = pd.qcut(
    rfm["Recency"],
    q=5,
    labels=[5, 4, 3, 2, 1]
)

rfm["F_Score"] = pd.qcut(
    rfm["Frequency"].rank(method="first"),
    q=5,
    labels=[1, 2, 3, 4, 5]
)

rfm["M_Score"] = pd.qcut(
    rfm["Monetary"],
    q=5,
    labels=[1, 2, 3, 4, 5]
)

rfm["RFM_Total"] = (
    rfm["R_Score"].astype(int)
    + rfm["F_Score"].astype(int)
    + rfm["M_Score"].astype(int)
)

# Customer segmentation
def segment_customer(score):
    if score >= 13:
        return "Champions"
    elif score >= 10:
        return "Loyal Customers"
    elif score >= 8:
        return "Potential Loyalists"
    elif score >= 5:
        return "At Risk"
    else:
        return "Lost Customers"

rfm["Segment"] = rfm["RFM_Total"].apply(segment_customer)
```

# Insight

* **Acquisition Satiation vs. Relationship Deficit:** The massive drop from Month 1 (100%) to Month 2 (23%) indicates that while the business has an exceptional marketing and onboarding mechanism that pulls in high initial transaction numbers, it lacks an automated engagement loop to secure immediate second purchases.
* **The December 2009 Blueprint:** The outsized success of the December 2009 cohort (35% Month 2 Retention) suggests specific seasonal items, promotional gift-giving mechanics, or early platform loyalty campaigns during that period were highly sticky. This specific operational playbook should be reverse-engineered for future winter campaigns.
* **The Leaky Revenue Bucket:** Because **"At Risk"** represents the single largest segment grouping on the financial page, a massive percentage of the platform's historical £17.37M revenue is at immediate risk of disappearing permanently. Retaining an existing customer is multiple times cheaper than acquiring a new one.

# Conclusions

1.  **Deploy an Immediate Month 2 Retargeting Sequence:** The single highest-leverage business objective is fixing the Month 2 drop-off cliff. The marketing team should immediately implement automated email flows, secondary purchase discount coupons, or loyalty points triggers explicitly targeting users exactly 14 to 30 days after their very first order.
2.  **Institutionalize RFM-Driven Segment Campaigns:** Instead of sending blast marketing communications to the entire database, use the data tables generated on Page 2 to split campaigns. Target the high-volume **"At Risk"** cohort with heavy re-engagement win-back incentives, while protecting the high-AOV **"Champions"** with exclusive product drops or VIP tier statuses.
3.  **Optimize High-Value Acquisition Channels:** Since the Average Customer Value is quite robust (£2,956), marketing spends should be heavily reallocated toward discovering channels that mimic the quality of the late-2009 shoppers rather than chasing low-cost, high-churn acquisition spikes that deplete operational resources without providing long-term retention.
